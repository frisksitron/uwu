# uwu — Unified Workspace Utility

A desktop terminal emulator with built-in project management. Built with Electron, Solid.js, TypeScript, and xterm.js.

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build:win    # Windows (NSIS)
pnpm build:mac    # macOS (DMG)
pnpm build:linux  # Linux (AppImage/snap/deb)
```
